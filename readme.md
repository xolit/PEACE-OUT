🎮 PEACE-OUT

A fast-paced 3D chase survival game inspired by Backdoor-style gameplay — but with a twist.

🧠 Overview

PEACE-OUT is a 3D survival runner where your only goal is simple:

Run. Escape. Survive.

An enemy is constantly chasing you, and every second matters. Unlike typical chase games, this one introduces a slightly different pacing and feel—making movement, awareness, and reaction time the key to survival.

⚡ Features
🏃‍♂️ Continuous Chase Mechanics – The enemy never stops hunting you
🎮 Smooth 3D Movement – Built for responsive controls
👁️ Immersive Perspective – Keeps tension high at all times
🧩 Simple but Addictive Gameplay – Easy to play, hard to master
🔄 Replay Value – Try to survive longer each run
🛠️ Tech Stack
🎯 Engine: Godot (3D)
💻 Language: GDScript
🎨 Assets: Custom / game-ready assets
🚀 Getting Started
1. Clone the repo
git clone https://github.com/xolit/PEACE-OUT.git
cd PEACE-OUT
2. Open in Godot
Open Godot Engine
Click Import Project
Select the project folder
3. Run the game
Press ▶️ Play inside Godot
🎮 Controls
Action	Key
Move Forward	W
Move Left	A
Move Right	D
Move Back	S
Look Around	Mouse
