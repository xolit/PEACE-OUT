## 🧠 Overview

**PEACE-OUT** is a 3D survival runner where your only goal is simple:

> **Run. Escape. Survive.**

An enemy is constantly chasing you, and every second matters.  
Unlike typical chase games, this one introduces a unique pacing and feel—making movement, awareness, and reaction time the key to survival.

---

## ⚡ Features

- 🏃‍♂️ **Continuous Chase Mechanics** – The enemy never stops hunting you  
- 🎮 **Smooth 3D Movement** – Built for responsive controls  
- 👁️ **Immersive Perspective** – Keeps tension high at all times  
- 🧩 **Simple but Addictive Gameplay** – Easy to play, hard to master  
- 🔄 **High Replay Value** – Try to survive longer each run
